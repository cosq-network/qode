# Java skill

## What it does
The Java skill handles compile/run actions, project builds, and test workflows for Java.

## Tools
Key tools:
- `java_compile_and_run`
- `java_project_build`

## How to use
Use it for:
- running a Java class,
- compiling sources,
- building Maven or Gradle projects.

## Example usage
Compile and run:
```text
Compile and run src/main/java/App.java.
```

Build a Maven project:
```text
Build the current Maven project and show test results.
```
