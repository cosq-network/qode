# Dotnet skill

## What it does
The dotnet skill is for .NET workflows including build, run, test, and diagnostics.

## Tools
It maps to tooling that runs `dotnet` commands for build and test flows of .NET projects.

## How to use
Use it for:
- building or running a .NET project,
- running filtered tests,
- checking runtime or SDK details.

**Examples**

- Build the current dotnet project.
- Run unit tests filtered by Category.
