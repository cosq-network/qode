# NPM skill

## What it does
The npm skill manages dependencies, scripts, audits, and outdated checks for Node.js projects.

## Tools
Key tools:
- `npm_run_script`
- `npm_list_dependencies`
- `npm_check_outdated`
- `npm_audit`

## Example usage
Run a script:
```text
Run npm test in the current project.
```

Check outdated:
```text
List outdated dependencies and suggest updates.
```

Audit:
```text
Run npm audit and summarize fixable issues.
```
