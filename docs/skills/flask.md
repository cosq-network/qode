# Flask skill

## What it does
The Flask skill handles Flask app creation, development server behavior, route inspection, and debug control.

## Tools
- `flask_create_app`
- `flask_run_server`
- `flask_routes_list`
- `flask_debug_enable`

## How to use
Use the Flask skill for Flask-specific tasks such as creating apps, starting the dev server, listing routes, and enabling debug mode.

Example prompt:
`Create a small Flask app and show how to run it.`

## When to use
Use it when you need Flask scaffolding, server execution, or runtime route inspection.
