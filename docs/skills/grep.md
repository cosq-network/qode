# Grep skill

## What it does
The grep skill supports search, replace, and batch-update workflows across project files.

## Tools
Key tools:
- `grep_regex`
- `grep_find_and_replace`
- `grep_stream_file_updates`
- `grep_search_then_replace`

## How to use
Use it when you need Qode to:
- search across file contents by pattern,
- find and replace text,
- apply repeated replacements with safeguards.

## Example usage
Search for TODOs:
```text
Find all files that contain TODO.
```

Replace a string:
```text
Replace console.log with logger.info across src/.
```
