# Flutter skill

## What it does
The Flutter skill supports mobile and cross-platform Flutter workflows such as build, test, dependencies, doctor checks, and clean.

## Tools
- `flutter_command(action='build'|'test'|'pub_get'|'doctor'|'clean', buildTarget?, extraArgs?)`

## Example usage
Run a dependency update:
```text
Run Flutter pub get and summarize changed package versions.
```

Run a health check:
```text
Run flutter doctor and describe any issues found.
```
