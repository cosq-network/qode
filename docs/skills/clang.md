# Clang skill

## What it does
The clang skill works with the native-build system for C/C++ code that should be compiled or analyzed through clang-style workflows.

## Tools
Clang-related behavior is surfaced through native-build tooling.

## How to use
Use the clang skill for:
- C/C++ compile flows that want clang diagnostics,
- analyzing build problems on macOS or Linux.

**Examples**

- Check this project with clang-style compile diagnostics.
- Analyze build errors in src/main.c and suggest fixes.
