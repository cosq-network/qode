# Native Build skill

## What it does
The native-build skill covers C/C++ build workflows using CMake, Make, GCC, Clang, and CTest.

## Tools
Key tools include:
- `cmake_configure`
- `cmake_build`
- `cmake_run_tests`
- `cmake_clean`
- `make_run`
- compilation and analyze actions exposed through native-build tooling

## How to use
Use it when you need Qode to:
- configure a native build,
- build tests,
- compile individual files,
- report build and compiler output.

Example prompt:
`Configure this native project with CMake for release.`
