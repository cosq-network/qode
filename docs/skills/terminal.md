# Terminal skill

## What it does
The terminal skill manages local shell execution, command history, and terminal sessions.

## Tools
Key tools:
- `terminal_exec_command`
- `terminal_shell_history_list`
- `terminal_create_terminal`
- `terminal_ensure_pty_ready`

## Example usage
List shell history:
```text
Show recent terminal history.
```

Run a command:
```text
Run the tests and show the last 50 lines of output.
```
