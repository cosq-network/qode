# Node skill

## What it does
The node skill runs Node.js files, checks runtime info, and can execute TypeScript transparently.

## Tools
Key tools:
- `node_run_file`
- `node_get_info`

## Example usage
Run a script:
```text
Run src/index.js and show stdout, stderr, and exit code.
```

Run TypeScript:
```text
Run src/cli.ts with arguments --help.
```

Check runtime:
```text
Show Node version and installed npm global packages.
```
