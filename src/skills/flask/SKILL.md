Use Flask tooling when the user asks to create, run, inspect, or debug a Flask app.
Preferred sources of truth:
- local registered tools: flask_create_app, flask_run_server, flask_routes_list, flask_debug_enable
Every flask action must map directly to one of those tools or another validated Flask workflow.
